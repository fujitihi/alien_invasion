# Alien Invasion
A small shooting game built with Python (Pygame).

## How to Play
- Move with arrow keys  
- Shoot with space bar
- Quit with 'q' key
- Avoid enemy and survive as long as you can!

## Features
- Multiple enemy waves  
- Score tracking    

## Tech Stack
- Python 3.x  
- Pygame  

## How to Run
```bash
pip install pygame
python alien_invasion.py
